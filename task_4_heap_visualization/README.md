# Завдання 4: Візуалізація бінарної купи

## Опис

Візуалізація структури бінарної купи (піраміди) за допомогою networkx та matplotlib.

## Реалізація

### Структура бінарної купи

**Властивості:**
- Мінімальна купа: батьківський вузол завжди менший за дочірні
- Повне бінарне дерево: всі рівні заповнені, останній рівень заповнюється зліва направо
- Зберігається як масив з індексацією: parent = (i-1)//2, left = 2*i+1, right = 2*i+2

### Операції

1. **insert(value)**: Додає елемент та просіює вгору
2. **extract_min()**: Видаляє мінімальний елемент та просіює вниз
3. **build_tree_from_heap()**: Побудова дерева для візуалізації

## Приклад візуалізації

### Вхідні дані
```
Елементи: [0, 27, 4, 1, 32, 5, 10, 3, 18, 19, 20]
```

### Структура купи

```
        0
      /   \
     1     4
    / \   / \
   3  27  5  10
  / \ / \
 18 19 32 20
```

### Кольорове кодування рівнів

| Рівень | Колір | Елементи |
|--------|-------|----------|
| 0 | Світло-блакитний | 0 |
| 1 | Світло-зелений | 1, 4 |
| 2 | Світло-жовтий | 3, 27, 5, 10 |
| 3 | Світло-рожевий | 18, 19, 32, 20 |

## Результати операцій

| Крок | Додано | Стан купи (перші 5) | Операція |

### Витягнення мінімуму

| Операція | Мінімум | Новий корінь | Просеювання |
|----------|---------|--------------|-------------|

| extract_min() | 1 | 3 | Просеювання вниз |
| extract_min() | 3 | 4 | Просеювання вниз |

## Властивості купи

### Підтримка структури

Після кожної операції купа зберігає властивість мінімальної купи:
- Корінь завжди мінімальний елемент
- Кожен батьківський вузол ≤ дочірніх вузлів

### Продуктивність операцій

| Операція | Складність | Опис |
|----------|------------|------|
| insert() | O(log n) | Просеювання вгору |
| extract_min() | O(log n) | Просеювання вниз |
| peek() | O(1) | Доступ до кореня |
| build_tree() | O(n) | Побудова дерева для візуалізації |

## Візуалізація

Візуалізація показує:
- **Структуру дерева**: Зв'язки між вузлами
- **Кольорове кодування**: Різні кольори для різних рівнів
- **Значення вузлів**: Числові значення в кожному вузлі
- **Геометричне розташування**: Позиції вузлів для кращого розуміння структури

1. **Коректність**: Купа правильно підтримує свою структуру після операцій

3. **Ефективність**: Операції виконуються за логарифмічний час
