# Завдання 2: Фрактал "дерево Піфагора"

## Опис

Рекурсивна побудова та візуалізація фрактала "дерево Піфагора" з можливістю вказати рівень рекурсії.

## Реалізація

### Алгоритм побудови

Фрактал будується рекурсивно:
1. Починаємо з базового квадрата
2. На верхній стороні квадрата будуємо прямокутний трикутник
3. На гіпотенузі трикутника будуємо два дочірні квадрати
4. Повторюємо процес для кожного дочірнього квадрата до досягнення максимального рівня

### Геометричні обчислення

Для кожного квадрата обчислюються:
- Координати вершин квадрата
- Позиція дочірніх квадратів на основі кута розгалуження (за замовчуванням 45°)
- Розмір дочірніх квадратів (зменшується на коефіцієнт √2)

## Результати візуалізації

### Таблиця кількості елементів за рівнями

| Рівень рекурсії | Кількість квадратів | Кількість трикутників | Загальна кількість елементів |
|-----------------|---------------------|----------------------|------------------------------|
| 0 | 1 | 0 | 1 |
| 1 | 3 | 1 | 4 |
| 2 | 7 | 3 | 10 |
| 3 | 15 | 7 | 22 |
| 4 | 31 | 15 | 46 |
| 5 | 63 | 31 | 94 |
| 6 | 127 | 63 | 190 |
| 7 | 255 | 127 | 382 |

**Формула:** Кількість квадратів = 2^(n+1) - 1, де n - рівень рекурсії

### Приклади візуалізації

**Рівень 2:**
- 7 квадратів
- 3 трикутники
- Проста структура, добре видно геометрію

**Рівень 4:**
- 31 квадрат
- 15 трикутників
- Детальна структура дерева

**Рівень 6:**
- 127 квадратів
- 63 трикутники
- Дуже детальна структура з багатьма гілками

## Режими візуалізації

### 1. Matplotlib (статична)
- Швидка побудова
- Можливість збереження у файл
- Підходить для аналізу структури

### 2. Turtle (анімація)
- Інтерактивна побудова
- Візуалізація процесу створення
- Підходить для демонстрації

## Параметри

- **Рівень рекурсії**: Контролює глибину дерева (рекомендовано 2-6)
- **Кут розгалуження**: За замовчуванням 45°, можна змінити
- **Початковий розмір**: Розмір базового квадрата

## Висновки

1. **Коректність**: Фрактал будується правильно згідно з геометричними правилами
2. **Рекурсія**: Ефективно використано рекурсивний підхід для побудови дерева
3. **Візуалізація**: Обидва режими (matplotlib та turtle) працюють коректно
4. **Масштабованість**: Алгоритм ефективно працює для різних рівнів рекурсії

**Спостереження:**
- Зі збільшенням рівня рекурсії кількість елементів зростає експоненційно
- Для рівнів > 7 візуалізація стає дуже складною через велику кількість елементів
- Оптимальний рівень для візуалізації: 3-5
